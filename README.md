# Expense Tracker

A simple and user-friendly Expense Tracker application built with Node.js and Express. This application helps users record, manage, and monitor their daily expenses efficiently.

## Features

* Add new expenses
* View all recorded expenses
* Store expense data locally using JSON
* Responsive and clean user interface
* Lightweight and easy to set up

## Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express.js
* Data Storage: JSON File

## Installation

1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/expensetracker.git
```

2. Navigate to the project directory:

```bash
cd expensetracker
```

3. Install dependencies:

```bash
npm install
```

4. Start the application:

```bash
npm start
```

5. Open your browser and visit:

```
http://localhost:3000
```
